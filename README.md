### Usage:
* **bf!help** - useless help
* **bf!run "input"code** - execute code

(**Note** that the code works without giving any input.)

Examples:
* bf!run ++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.
	* Output: **Hello World!**
* bf!run "Hello World!">,[>,]<[<]>.[>.]
	* Output: **Hello World!**
